مۇقەددەس ئابدۇروسۇل


26ياشلىق شېرىنئاينىڭ ئائىلىسى نامرات بولغاچقا، ئۈرۈمچىگە كېلىپ جىق پۇل تاپماقچى بولدى، ئۇنىڭ ئۈرۈمچىگە كەلگىنىگىمۇ ھەش-پەش دېگۈچە ئىككى يىلدىن ئاشتى. بۇي جەرياندا بىر قانچە تېز تاماقخانىلاردا مۇلازىم بولدى، تېز تاماقخانىلارغا چىرايلىق ياسىنىپ كىرىپ-چىققان قىزلارنى كۆرگىنىدە، ئۆزىنىڭمۇ ئاشۇلاردە چىرايلىق ياسىنىپ ئېسىل يەرلەرگە بارغۇسى كەلدى. شۇنىڭ بىلەن شېرىنئاي ئىشلەۋاتقان تېز تاماقخانىدىن چقىپ كېتىپ، بەيگە مەيدانى ئەتىراپىدا بىر ئۆينى ئىجارىگە ئېلىپ ئولتۇردى. بۇ جەرياندا ئۇ ئۆزىگە ئوخشاش يا قا يۇرتلۇق گۈلنىسا بىلەن تونۇشۇپ قېلىپ، ئىككىسى بىر ئۆينى ئىجارىگە ئېلىپ ئولتۇردى. ئۇلار تېز تاماقخانىدا ئىشلىسە ئايدا ئاران نەچچە يۈز يۈەنلا پۇل تاپىدىغانلىقىنى، شۇڭا جاپا تارىتمايلا كۆپرەك پۇل تاپىدىغان ئىشنى قىلىشنى، جىقراق پۇل تېپىۋېلىپ شەھەرلىك قىزلاردىنمۇ پوزۇر يۈرۈشنى مەسلىھەتلەشتى.
بىر كۈنى گۈلنىسا شېرىنئاينىڭ يېنىغا كۈلۈپ كىردى:
-ئاداش، تېز بولە ئورنۇڭدىن تۇرۇپ تېزرەك ئۆزەڭنى تۈزەشتۈرگىن، پۇل تاپىدىغان ئىش چىقتى. گۈلنىسانىڭ گېپىنى چۈشىنەلمىگەن شېرىنئاي سوئال نەزىرى بىلەن گۈلنىساغا قارىدى.
-قارا ئاداش، بىكار ئولتۇرساقمۇ ئولتۇرىمىز، ئۇنىڭدىن كۆرە چەتئەللىكلەر بىلەن ھاراق ئىچىشىپ ئولتۇرۇشۇپ بەرسەك نەچچە سائەتنىڭ ئىچىدىلا ئاشخانىدا ئىشلىگەن بىر ئايلىق پۇلىمىزنى تاپالايمىز، بۇنىڭدىنمۇ ئاسان ئىش بارمۇ؟-دېدى گۈلنىسا.
-ئەگەر بىزنىڭ چەتئەللىكلەر بىلەن يۈرگىنىمىزنى تونۇشلىرىمىز ياكى باشقىلار كۆرۈپ قالسا نىمە دەيمىز؟-دېدى شېرىنئاي.
-ۋىيەي، نىمە دەيىتتۇق، تىل ئۆگىنىۋاتىمىز دەيمىز. ھازىر چەت ئەل تىلى ئۆگىنىۋاتقانلار جىققۇ. قارا، جىقراپ پۇل تېپىۋېلىپلا يۇرتقا كېتىپ ياخشىراق بىرەرسىنى تېپىپ تويىمىزنى قىلىۋېرىمىز ئاداش، ھازىرقى ئوغۇللارچۇ چىرايلىق كىيىنىپ يۈرسەك، پۇلىمىز بولسا،(مەندىن قوشۇمچە: باشقا پاكلىق،ئىپپەت،ۋىجدان،غۇرۇر...ئېتىقاد) بومىسىمۇ بىزنى ئالماي قالامتى، ئارتۇق ئىشلارنى ئويلاپ كەتمە...
گۈلنىسا بىلەن شېرىنئاي مەلۇم بىر مېھمانىخانىنىڭ ياتىقىدا بىر -قانچە چەتئەللىكلەرگە ھەمراھ بولۇپ ھاراق ئىچىشىپ،تۇنجى قېتىملىق سودىنى ئوڭۇشلۇق پۈتتۈردى. شۇنىڭدىن باشلاپ شېرىنئاي بۇ سودىنىڭ ئېپىگە كىرىۋالدى. ئۇ چەت ئەللىكلەرگە ھەمراھ بولۇپ ئىچىپ - چېكىپ ئويناش جەريانىدا باشقىلارغا ( تەپ تارتماستىن ) چەتئەل تىلى ئۆگىنىۋاتىمەن ، دەپ ئۆز سەنىمىگە ( قانجۇق يورغىسىغا ) دەسسىدى.
شېرىنئاي بۇ يىل 3 -فېۋرال ئادىتى بويىچە شەھىرىمىزدىكى مەلۇم مېھمانساراينىڭ ئالدىدا كىرىپ - چىقىۋاتقانلارنى ( شەھۋانىي بۇزۇق كۆزلىرى بىلەن ) كۈزىتىپ تۇردى. شېرىنئاي كۆڭلىدە بۈگۈن پۇلى جىقراق ( ئولجىنى ) بىرسىنى تاپسام ( ئۇنىڭغا ئاياق-ئاستى بولۇپ ياتسام) بولاتتى ، دەپ ئويلاپ تۇرىشىغا ئالدىدا پاكىستانلىق سەدۇللا 45 بەش ياش (مەينەت شۆلگىيىنى ئېقىتىپ)پەيدا بولدى. ( ياپانا ئاللاھ ئۆزەڭ بار...بۇ بىر بىرىگە يات ئىككى چەت ئەللىك) ناھايتى تېزلا تونۇشۇپ(ئەر - خوتۇندەك) چىقىشىپ كەتتى -دە، بىر -بىرىگە تېلفۇن نومۇرلىرىنى قالدۇرۇشتى. شۇ كۈنى كەچتە سەدۇللا شېرىنئايغا تېلفۇن بېرىپ ئىجارىگە ئالغان ئۆيىگە ھازىرلا كېلىشىنى ئېيىتتى. ئارىدىن ئۇزاق ئۆتمەيلا شېرىنئايمۇ ( ئىت ئىللىماس ) قونچاقتەك ياسىنىپ سەدۇللانىڭ ئۆيىگە يېتىپ كەلدى. ئۇلار ئۆيدە مۇزىكا ئاڭلىغاچ بىر قانچە بوتۇلكا پىۋا ئىچكەندىن كېيىن، سەدۇللا شېرىنئايغا ئۆزىنى ( ھايۋانىي نەپىسىنى قاندۇرۇپ ) ئوبدان رازى قىلسىلا پۇلدا چاتاق يوقلۇقىنى ئېيىتتى. كېچە سائەت2:00 دىن ئاشقاندا شېرىنئاي سەدۇللانىڭ ئۇخلاپ (لاي بۇتتەك ) قېتىپ كەتكەنلىكىنى پەملەپ ئاستا ئورنىدىن تۇردى-دە، كىيىملىرىنى شەپە چىقارماي كىيىپ،( بۇنداق ھايۋانغا كىيىم كىيىپ نەكەپتۇ ) سەدۇللانىڭ 19مىڭ 547يۈەن قىممىتىدىكى ( سېسىق ) پۇل قاتارلىق نەرسىلىرىنى ئوغۇرلاپ تۇيدۇرماستىن چىقىپ كەتتى.
سەدۇللا تاڭ سەھەرگە يېقىن ئويغىنىپ شېرىنئاينىڭ يوقلۇقىنى، يانچۇقىدىكى دوللار ۋە تېلفۇننىڭ يوقاپ كەتكەنلىكىنى كۆرۈپ،ساقچىغا دېلو مەلۇم قىلدى.
تەڭرىتاغ رايونلۇق ج خ ئىدارىسىنىڭ جىنايى ئىشلار 3 - چوڭ ئەتىرىتىدىكى رازۋېتچىكلار سەدۇللا تەمىنلىگەن يىپ ئۇچىغا ئاساسەن رازۋېتكا قىلىپ، 13 - فېۋرال شېرىنئاينى دۆڭكۆۋرۈكتە ( رەسۋالارچە ) قولغا چۈشۈردى. شېرىنئاي سوراق جەريانىدا، سەدۇللانىڭ ئالدىن دېيىشكەن پۇلىنى بەرمىگەنلىكى ئۈچۈن پۇللىرىنى ئوغىرلىغانلىقىنى ئىقراق قىلدى.
شۇنىڭ بىلەن شېرىنئاي 13 - فېۋرال جىنايى ئىشلار بويىچە توختىتىپ قويۇلۇپ، 18 - مارت تەڭرىتاغ << تىيانشان>> رايونلۇق تەپتىش مەھكىمىسى تەرىپىدىن ئوغرىلىق جىنايىتى بىلەن قولغا ئېلىشقا تەستىقلاندى
